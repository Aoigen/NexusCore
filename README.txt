    Nexus Core

O Nexus Core é um projeto desenvolvido com o objetivo de explorar e consolidar os fundamentos da
programação em Java, utilizando o mínimo possível de bibliotecas externas e priorizando a
implementação manual de funcionalidades.

O projeto está em desenvolvimento constante, recebendo mudanças em suas características, objetivos e tecnologias utilizadas.

    Objetivos principais:
- Servir como base para o aprendizado de frameworks e ferramentas em projetos futuros.
- Demonstrar habilidades no uso de tecnologias como Java, MySQL (Workbench), HTML, CSS e JavaScript.
- Desenvolver um sistema CRUD funcional com foco em simplicidade e organização de código.

    Características do projeto:
- Arquitetura enxuta: estrutura organizada para facilitar o aprendizado e o desenvolvimento incremental.
- Código otimizado: sem dependências desnecessárias, reforçando a compreensão de conceitos fundamentais.
- Integração com MySQL: suporte ao armazenamento de dados de forma eficiente e segura.
- Interface Web: formulários HTML estilizados com CSS e interação básica em JavaScript.

O Nexus Core é ideal para quem busca um ponto de partida sólido em Java e desenvolvimento web, com ênfase na compreensão de conceitos essenciais antes de explorar ferramentas mais complexas.

    Tecnologias utilizadas:
- Java: lógica e funcionalidade principal.
- Jakarta Servlet API: para a criação de uma aplicação web dinâmica.
- MySQL: banco de dados para armazenamento de informações.
- HTML, CSS e JavaScript: interface simples e funcional.
